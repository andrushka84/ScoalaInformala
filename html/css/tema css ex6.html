An example using non-semantic markup

This is one of two sample pages created to demonstrate the difference between pages using semantic HTML markup and those who don't.

What you see isn't always what you get

If you compare this page with the one using semantic markup, you won't see significant differences in your browser.

But what's on the inside can make a world of difference for accesibility, search engine optimization and workflow. Read the accompanying blog entry for more.
